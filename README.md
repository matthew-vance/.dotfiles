# dotfiles

`stow` is used to manage the symlinks. It is available in most package managers.
This whole thing basically works by mirroring the layout of the home directory and symlinking all of the files.

## Requirements

- `stow`
- `git`
- `make`
- `zsh`
- `tmux`
- `neovim`
- `fzf`
- `ripgrep`
- `bat`
- `fnm`
- `exa`

## Usage

```sh
make install
```

```sh
make uninstall
```
